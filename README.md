 Bash script para instalar ferramentas importantes em um computador limpo com Ubuntu/_Debian-based_ Linux
## Ferramentas:
 - NVM
 - NPM/Node
 - git
 - gulp
 - LAMP
 - Composer
 - Laravel
 - Yii2
 - pipenv/Python 3
 - OpenJDK
 - RVM
 - VSCode
 - Ruby
